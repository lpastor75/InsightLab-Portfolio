# Formula 1 Performance Analytics

Análisis de rendimiento histórico de Fórmula 1 utilizando Python para procesar y visualizar datos de carreras y pilotos.

## 📌 Objetivo

Este proyecto explora los datos históricos de la Fórmula 1 para identificar patrones de rendimiento, tendencias de equipos y desempeño de pilotos en diferentes temporadas y contextos.

## 📁 Estructura del Proyecto

- `data/` — Contiene los datasets CSV utilizados para el análisis.  
- `formula1_performance_analysis.ipynb` — Notebook Jupyter principal con todo el análisis.  
- `images/` — Carpeta con gráficos y visualizaciones exportadas durante el análisis.  
- `README.md` — Este documento.

## 📊 Tecnologías

- Python  
- Pandas, NumPy  
- Matplotlib / Seaborn  
- Jupyter Notebook  

## 🧠 Descripción de Datos

Los datos se componen de múltiples archivos CSV que representan resultados de carreras, posiciones de pilotos, tiempos por vuelta, paradas en boxes y clasificaciones. Estos datos permiten realizar análisis estadísticos y visualizaciones profundas del rendimiento en distintas épocas de la F1.

## 📈 Resultados Clave

- Visualizaciones del rendimiento de pilotos a lo largo de las diferentes temporadas de Fórmula 1.  
- Comparación entre equipos y pilotos basada en métricas de desempeño histórico.  
- Análisis de consistencia en tiempos de vuelta y estrategias de carrera.
